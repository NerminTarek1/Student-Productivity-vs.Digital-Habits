# Student-Productivity vs. Digital Habits

Dataset: https://www.kaggle.com/datasets/sehaj1104/student-productivity-and-digital-distraction-dataset 
Project Overview

This project investigates the relationship between students' digital habits — including social media usage, phone screen time, YouTube consumption, and gaming — and their overall academic productivity. Using a dataset of 20,000 student records with 18 features, The project investigates how these behavioral factors interact and influence student productivity.

Objectives

•	Perform a complete 8-step Exploratory Data Analysis (EDA) pipeline on the dataset.
•	Identify the key digital and lifestyle factors that positively or negatively affect student productivity.
•	Segment students into behavioral profiles.
•	Deliver an interactive dashboard summarizing findings across 5 analytical dimensions.
•	Explore relationships between sleep, stress, study habits, and productivity.
•	Design interactive dashboards using Tableau, Excel and Power BI.
•	Case studies using SQL.

Tools & Technologies

•	Python (Pandas, NumPy, Matplotlib, Seaborn)
•	Tableau
•	Excel
•	Power BI
•	SQL
